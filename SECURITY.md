# Security Policy

## Scope

This repository contains **public documentation only**.

It does not include:
- Executable code
- Backend services
- APIs
- Infrastructure configuration
- Credentials or secrets

As such, there are no direct security risks associated with this repository itself.

---

## Reporting security issues

If you believe you have discovered a security issue related to the **LinguaVerseSchool platform**, please do **not** open a public GitHub issue.

Instead, report it privately via email:

support@linguaverseschool.com

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Any relevant screenshots or logs

---

## Responsible disclosure

Responsible disclosure is appreciated.  
Security reports will be reviewed and addressed as appropriate.

Thank you for helping keep LinguaVerseSchool secure.
